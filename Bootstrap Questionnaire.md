## What Happens Next

Once you've filled this out:

1. Connect the **Indeed**, **Dice**, and **ZipRecruiter** connectors in the Claude app (Settings → Connectors, or ask Claude to suggest them) and install the **Claude in Chrome** extension for LinkedIn
2. Open Claude desktop and start a new Cowork session
3. Share this filled questionnaire with Claude
4. Say: *"Please use my answers to create my about-me.md file and set up my workspace using the multi-board job search framework."*

Claude will build all your memory files, create your base resumes, and set up the daily scan in one session.
